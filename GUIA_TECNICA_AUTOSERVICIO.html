Manual de Usuario - Sin Soporte

Este documento contiene toda la información técnica necesaria para que usted mismo personalice y publique este template. Al adquirir este producto, el comprador asume la responsabilidad total de su edición.

AVISO IMPORTANTE: No se brinda soporte técnico personalizado. Si usted daña el código o las rutas de los archivos, deberá descargar nuevamente el archivo original de la plataforma de compra.

1. Edición de Textos y Datos

Para cambiar nombres, descripciones o precios, abra los archivos .html con un editor de texto (se recomienda Visual Studio Code o Sublime Text).

  • Precios: Busque la etiqueta $15.00 y cámbiela por su valor local.
  • Dirección: Localice el texto dentro del <footer> al final de cada página.
  • WhatsApp: Busque el enlace href="https://wa.me/1234567890" y reemplace los números después del / con su número (incluya código de país, sin el signo +).

2. Cambio de Imágenes y Logo

Todas las imágenes se encuentran en la carpeta /images. Para que el sitio no se rompa, debe respetar lo siguiente:

  • Logo: Su nuevo logo debe llamarse exactamente logo barber.png y ser de formato PNG transparente.
  • Fotos de Galería: Si reemplaza una foto, asegúrese de que la nueva tenga las mismas dimensiones (ejemplo: 800x600px) para mantener la alineación.
  • Nombres de archivos: No use espacios ni tildes en los nombres de las fotos (ejemplo: use corte_moderno.jpg, NO corte moderno.jpg).

3. Publicación (Hosting)

Usted puede subir estos archivos a cualquier hosting que soporte HTML estático:

  1. Suba el contenido completo del ZIP a la carpeta raíz de su servidor (usualmente llamada public_html o www).
  2. Asegúrese de que el archivo index.html esté en la carpeta principal.
  3. Verifique que las carpetas /css, /images y /JavaScript se hayan subido correctamente.

4. Resolución de Problemas (FAQ)

  • ¿Por qué no se ven las fotos? Verifique que no haya movido las fotos fuera de la carpeta /images.
  • ¿Por qué no cargan los estilos (fondo blanco)? Asegúrese de no haber borrado la carpeta /css.
  • ¿El botón de WhatsApp no abre? Revise que el enlace empiece por https://wa.me/ seguido de su número completo.
