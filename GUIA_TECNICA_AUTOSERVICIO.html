Manual Maestro de Configuración

Este documento es la guía definitiva para personalizar cada rincón de su nueva plantilla web. Está diseñado para que usted tenga el control total del sitio, desde los textos básicos hasta la lógica interna de los formularios.

REGLA DE ORO: Antes de realizar cualquier cambio, cree una copia de seguridad de la carpeta original. No se brinda soporte técnico personalizado; este manual contiene todas las respuestas necesarias. Se recomienda usar Visual Studio Code para editar los archivos.

1. Estructura del Proyecto

Para que el sitio funcione, los archivos deben mantener esta organización. Si mueve un archivo de carpeta, los enlaces o imágenes dejarán de funcionar:

/Proyecto-Barberia
├── index.html (Inicio)
├── Contactanos.html (Citas y Mapa)
├── Nosotros.html, Galeria.html (Páginas secundarias)
├── /css (Estilos, colores y fuentes)
├── /images (Todas las fotografías y logos)
├── /JavaScript (Lógica de envío a WhatsApp)
└── /logos (Logos de marcas aliadas)

2. Configuración de WhatsApp (Lo más importante)

Todo el sistema de envío de mensajes (Reserva de Citas y Formulario de Contacto) se configura en un solo lugar.

  • Abra el archivo JavaScript/main.js.
  • Busque la línea: const telBarberia = "593981766228";
  • Reemplace el número entre comillas por el suyo (Código de país + número, sin el símbolo +).
  • Formato de mensaje: En el mismo archivo, puede editar el texto que le llegará a su celular. El código %0A representa un salto de línea (Enter). No lo borre si desea mantener el orden del mensaje.

3. Personalización del Formulario de Citas

A. Agregar o Quitar Barberos

En Contactanos.html, busque el bloque de "Elige a tu Barbero". Para añadir uno nuevo, copie y pegue este código:

<div class="col-md-6">
  <input type="radio" class="btn-check" name="barbero" id="barb3" value="NOMBRE NUEVO">
  <label class="btn btn-reserva-transparente w-100 p-3" for="barb3">
    <i class="bi bi-person-fill me-2"></i> NOMBRE NUEVO
  </label>
</div>

Atención: El id="barb3" y el for="barb3" deben ser iguales entre sí, pero diferentes a los de los otros barberos (use barb4, barb5, etc.).

B. Cambiar Opciones de Asunto y Comentarios

Busque la etiqueta <select> en el formulario de contacto. Para cambiar una opción, edite el texto dentro de <option>:

<option value="Consulta General">Consulta General</option>

4. Gestión de Servicios y Productos

A. Editar o Eliminar Secciones

Cada sección (Servicios, Nosotros, Galería) está envuelta en etiquetas <section>. Para eliminar una, borre desde la etiqueta de apertura hasta la de cierre. Para agregar una, duplique una sección existente.

B. Cambiar Productos de Venta

En Venta de Productos.html, cada producto es una "Card". Usted puede:

  • Cambiar Imagen: Reemplace el nombre en src="images/producto.jpg".
  • Cambiar Precio: Busque el texto con el símbolo $ y edítelo.

5. Guía de Imágenes y Mapa

Elemento Ubicación Tamaño / Formato
Logo Principal /images/logo barber.png PNG transparente (250x150px)
Fotos de Cortes /images 600x600px (Cuadradas)
Mapa Google Contactanos.html Pegue el link de "Insertar Mapa" en el src del iframe.

6. Colores, Fuentes y Estilos (Avanzado)

Si desea cambiar el color dorado del sitio, abra css/style.css y localice la sección :root al principio del archivo:

--color_primario: #F59E0B; /* Cambie este código hexadecimal por su color */
--color_secundario: black; /* Color de fondos y menús */

Para cambiar la tipografía global, modifique la línea: *{ font-family: "lora"; } por la fuente de su preferencia cargada desde Google Fonts.

7. Publicación Final

Para subir el sitio a internet:

  1. Contrate un servicio de Hosting y Dominio.
  2. Suba el contenido de la carpeta del proyecto a la carpeta public_html de su servidor.
  3. Asegúrese de que el archivo index.html esté en la raíz, no dentro de subcarpetas adicionales.
